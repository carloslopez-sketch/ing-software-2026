# Reflexión laboratorio 1

## Diferencias observadas con Copilot

Los comentarios influyen en las sugerencias generadas por Copilot.  
Cuando el comentario es más claro, el código generado es más preciso.  
Cuando es más general, las sugerencias son más simples.

## Comparación en Claude.ai

El prompt pobre genera respuestas menos útiles.  
El prompt efectivo produce código más claro y completo.  
Conclusión: un buen prompt mejora significativamente el resultado.
